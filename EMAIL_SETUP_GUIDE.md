# 📧 Настройка отправки Email

## ✅ Email интеграция настроена!

Теперь коды восстановления пароля отправляются на реальный email пользователя через Gmail SMTP.

## 🔧 Что нужно настроить:

### 1. Укажи свой Gmail в appsettings.json

Открой файл: `backend/EventRecommendationSystem.API/appsettings.json`

Найди секцию `EmailSettings` и **замени** `"SenderEmail"` на свой email:

```json
{
  "EmailSettings": {
    "SmtpServer": "smtp.gmail.com",
    "SmtpPort": 587,
    "UseSsl": true,
    "SenderEmail": "твой-email@gmail.com",  // ← ИЗМЕНИ ЭТО
    "SenderPassword": "cejx lony kiuo dlnu",
    "SenderName": "Event Recommendation System"
  }
}
```

**Важно:** 
- `SenderEmail` - это твой Gmail адрес
- `SenderPassword` - уже указан твой App Password от Google
- `SenderName` - имя отправителя (можно изменить)

### 2. Установи пакет MailKit

При первом запуске нужно восстановить NuGet пакеты:

```bash
cd backend
dotnet restore
```

Это автоматически скачает MailKit 4.3.0

### 3. Перезапусти бэкенд

```bash
cd backend/EventRecommendationSystem.API
dotnet run
```

## 🎯 Как это работает:

### Восстановление пароля:

1. **Пользователь** нажимает "Забыли пароль?" на странице входа
2. **Вводит email** и нажимает "Получить код"
3. **Бэкенд:**
   - Генерирует 6-значный код (например: 456789)
   - Выводит код в консоль (для отладки)
   - Отправляет красивое HTML письмо на email пользователя
4. **Пользователь:**
   - Проверяет почту (включая спам!)
   - Вводит код из письма
   - Устанавливает новый пароль
5. **Готово!** Можно входить с новым паролем

### Пример письма:

```
Тема: Восстановление пароля - Event Recommendation System

Здравствуйте, username!

Вы запросили восстановление пароля.

Ваш код восстановления:
    456789

Код действителен в течение 15 минут.

⚠️ Если вы не запрашивали восстановление, проигнорируйте это письмо.
```

## ⚠️ Возможные проблемы:

### "Ошибка отправки письма"

**Проверь:**

1. **Правильный ли email в SenderEmail?**
   ```json
   "SenderEmail": "твой-настоящий-email@gmail.com"
   ```

2. **Правильный ли App Password?**
   - Пароль должен быть из Google App Passwords
   - Формат: `xxxx xxxx xxxx xxxx` (с пробелами или без)
   - Проверь здесь: https://myaccount.google.com/apppasswords

3. **Включена ли 2FA в Google?**
   - App Passwords работают только с включенной 2FA
   - Включи здесь: https://myaccount.google.com/security

4. **Правильные ли настройки SMTP?**
   ```json
   "SmtpServer": "smtp.gmail.com",
   "SmtpPort": 587,
   "UseSsl": true
   ```

### Письмо не приходит

1. **Проверь спам** - Gmail может отправить первое письмо в спам
2. **Проверь логи бэкенда** - там будет код на случай проблем с email
3. **Подожди 1-2 минуты** - иногда есть задержка
